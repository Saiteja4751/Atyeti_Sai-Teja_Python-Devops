pushed into a Github repo using the GIT
-----------------------
1.git init
2.git add . (add all files at a time)
3.git commit -m 'commit the files'
4.git branch -M main
5.git remote add origin github-repo
6.git push -u origin main (pushing into a repo)


BRANCHING STRATEGIES
----------------------
git branch -m sai
git chekout <branch-name>
git merge main (mergeing sai with the main)

------
